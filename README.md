Web Teleprompter
A lightweight, browser‑based teleprompter designed for smooth reading, fullscreen presentation, and mirrored display.
Built with pure HTML, CSS, and JavaScript, it runs on any device — iPhone, Android, tablet, or desktop — without installation.

Features
Editable text input for preparing your script

Mirrored display (scaleX(-1)) for use with physical teleprompter glass

Automatic scrolling with multiple speed levels

Adjustable font size without losing the current scroll position

Fullscreen mode compatible with mobile and desktop browsers

Auto‑hide controls with touch/mouse detection

Hidden scrollbars for a clean reading experience

Responsive layout optimized for large text and long scripts

How It Works
Type or paste your script into the text box

Press the fullscreen button to start

Use the controls to adjust speed and font size

Tap or move the mouse to show the controls again

Press the fullscreen button again to exit

Technologies
HTML for structure

CSS for layout, mirroring, and scrollbar styling

JavaScript for scrolling logic, fullscreen handling, and UI behavior

Purpose
This project was created to provide a simple, accessible teleprompter that works anywhere — especially on mobile devices — without requiring apps, accounts, or downloads.
